# 나눔바른고딕 웹폰트

네이버에서 2013년 10월 한글날등을 기념하여 만든 나눔바른고딕을 웹폰트로 제작하였습니다.
제작 의도는 제 개인적인 사용이 목적이 었으며, 구글 웹폰트등 정식 배포 서비스나 나오면 그것을 사용해 주시기 바랍니다.

## 시작하기

나눔바른고딕 웹폰트의 깃허브 호스팅은 묶음형태의 CSS파일로 제공되며 3가지 형식으로 사용할수 있습니다.
